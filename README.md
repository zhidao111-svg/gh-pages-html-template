# NMIXX Macau 2025 Fan Concert Record Site
This repo is forked from gh-pages-html-template, rebuilt as my personal non-profit fan memory website.
Live preview: https://zhidao111-svg.github.io/gh-pages-html-template/

## Project Intro
Pure native HTML + CSS static page, built to archive NMIXX 2nd Fan Concert Macau fan-cam videos, personal concert experience and photo gallery.
100% open source, only for front-end learning & fan sharing, no commercial profit.

## Tech Stack
- HTML5
- Internal CSS styling
- GitHub Pages auto deployment
- MIT License open source

## Template Original Guide (from forked source)
A simple template for deploying an HTML website to GitHub Pages

### Instructions
1. Fork this repository
2. Clone the forked repository
3. Copy HTML files to the root of the cloned repository and overwrite the index.html file
4. Change the URL in CNAME to your own domain. If you don't have a domain, delete the CNAME file.
5. Commit the changes and push them to GitHub
6. Go to repo Settings -> Code and automation -> Actions -> General -> Workflow permissions -> Read and write permissions.
7. Go to repo Settings -> Code and automation -> Pages -> Build and deployment -> Branch -> gh-pages -> Save.
8. The website will be deployed to your GitHub Pages repository.
