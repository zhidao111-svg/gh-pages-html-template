# NMIXX Macau 2025 Fan Concert Record Site
This repository is forked from the base gh-pages-html-template, fully rebuilt into my personal non-profit fan memory static website.
Live website preview: https://zhidao111-svg.github.io/gh-pages-html-template/

## Project Introduction
Built with native HTML5 + internal CSS only.
This page archives fan-cam stage videos, my personal concert experience notes, and on-site photos for NMIXX's 2nd Fan Concert held in Macau on June 28, 2025.
The code is fully open-source under MIT license, only for front-end beginner practice and fan community sharing — zero commercial use or profit.

## Original Template Setup Guide (Fork Source Instructions)
A simple template for deploying an HTML website to GitHub Pages

### Instructions
1. Fork this repository
2. Clone the forked repository
3. Copy HTML files to the root of the cloned repository and overwrite the index.html file
4. Change the URL in CNAME to your own domain. If you don't have a domain, delete the CNAME file.
5. Commit the changes and push them to GitHub
6. Go to repo Settings -> Code and automation -> Actions -> General -> Workflow permissions -> Read and write permissions.
7. Go to repo Settings -> Code and automation -> Pages -> Build and deployment -> Branch -> gh-pages -> Save.
8. The website will be deployed to your GitHub Pages repository. The URL should look like https://username.github.io/gh-pages-html-template/
